# Nuevas Funcionalidades.
A diferencia de ser un plano mas grande de lo requerido, se creo un sistema en el cual no solamente sea un simple ordinario parqueadero universal, donde cualquier vehiculo se estaciona no. Se creo tipos de parqueaderos, como lo son el normal, el de gente discapacitada, el de las bicicletas y el de las motos, cada una de ellas con una tarifa diferencial. Además, el espacio no universal, es decir, el que tiende a ser reservado por ese vehiculo, solamente se puede poner a ese vehiculo en ese parqueadero. Creando asi unas restricciones en el parqueadero, tambien se puso el estacionamiento universal donde cada uno de los vehiculos que hay pueden estacionarsen ahi sin ningun tipo de restriccion. Se diseño y se considero necesario para la creatividad y auntenticidad del trabajo. 

# Automatizacion de Pintar la Matriz
He creado dos funciones de tipo lambda llamadas PintarEnHorizontal() y PintarEnVertical().
Las dos se encargan de Pintar una por medio de loop y otra de forma normal (1x1), una fila o una columna de manera instantanea, considerandose necesaria para el ahorro de tiempo y de lineas de codigo, siendo mas optimizado. Su implementacion se llevo a cabo en la funcion Generar_PMapa().
